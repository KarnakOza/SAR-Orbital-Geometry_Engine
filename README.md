# SAR-Orbital-Geometry-Engine
Satellite Orbit simulation, Radar observation geometry, SAR Imaging parameters

This project implements a simplified Synthetic Aperture Aperture Radar (SAR) orbital geometry simulator in C.

# Features
-----------------
• Orbital velocity computation
• Orbit period calculation
• Satellite position in ECI coordinates
• Earth rotation transformation (ECI -> ECEF)
• Radar look vector
• Slant range calculation
• Doppler centroid estimation
• Incidence angle calculation


# Applications
-----------------

This geometry engine models the observation physics used in SAR satellites such as Sentinel-1 and other Earth observation missions.

# Technologies
------------------

• C language

• Vector math library

• orbital mechanics

• Radar geometry modeling
